# Huerto

### Base

Control local/remoto de riego según las condiciones atmosféricas


#### Materiales
* Controlador
* Sensores
  * luz
  * temperatura
  * humedad
  * humedad del suelo
  * lluvia
* Wifi
* Placa de reles (para las electrobombas o electroválvulas)
* Electrobombas o Electroválvulas
* Alimentación
* Cables
